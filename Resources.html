Welcome to The Coding Atelier Code Kit

This interactive kit is designed for curious young creators ready to explore the world of coding. Scroll down to play, build, and remix — no experience needed!

Draw with Blockly Turtle

Use block coding to move a turtle and draw cool shapes! Great for learning loops and angles.

✨ Try this: Make the turtle draw a square, triangle, and circle!

Play Rodocodo

This game teaches coding through fun levels and instant feedback. Just click and play — no signup needed!

✨ Try this: Complete 3 levels and earn your certificate!

Blockly Maze Challenge

Help the character reach the goal by arranging blocks in the right order. It’s like solving a puzzle with code!

✨ Try this: Can you solve the maze in fewer blocks?

Guide the Bird with Blockly

Use block coding to help the bird reach its worm! Learn sequencing and logic in a fun way.

✨ Try this: Complete all 10 levels without crashing into a tree!
